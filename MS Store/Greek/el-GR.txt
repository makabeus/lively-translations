Περιγραφή:
Δωρεάν εφαρμογή ανοιχτού κώδικα για τον ορισμό GIF, βίντεο και Ιστοσελίδες ως ταπετσαρία της επιφάνειας εργασίας σας.
Με την υποστήριξη του προγράμματος περιήγησης Chromium και του προγράμματος αναπαραγωγής MPV, οι ταπετσαρίες σας υποστηρίζουν τις τελευταίες τεχνολογίες ιστού και πρότυπα βίντεο.

Αυτό είναι ένα έργο ανοιχτού κώδικα και η συμμετοχή της κοινότητας είναι ευπρόσδεκτη. Για να συμμετάσχετε παρακαλώ επισκεφθείτε
https://github.com/rocksdanister/lively

Χαρακτηριστικά:
Η αναπαραγωγή ταπετσαρίας διακόπτεται (χρήση πόρων 0%) κατά την εκτέλεση παιχνιδιών/εργασίας.
Υποστηρίζει πολλές οθόνες οποιουδήποτε λόγου διαστάσεων και ανάλυσης.
Αναπαραγωγή βίντεο με επιτάχυνση υλικού που υποστηρίζεται από το πρόγραμμα αναπαραγωγής MPV, ορίστε οποιοδήποτε βίντεο ως ταπετσαρία.
Με την υποστήριξη του Chromium Embedded Framework (CEF), ορίστε οποιονδήποτε ιστότοπο ως ταπετσαρία.
Χρησιμοποιήστε το Lively Audio API για να δημιουργήσετε ταπετσαρίες που αντιδρούν στον ήχο του συστήματος
Χρησιμοποιήστε το Lively System API για να δημιουργήσετε widget που εμφανίζουν πληροφορίες συστήματος.
Υποστηρίζει διαδικτυακές ροές βίντεο και συνδέσμους Shadertoy.com.
Πλήρως ανοιχτού κώδικα και δωρεάν, χωρίς χαρακτηριστικά πίσω από το paywall.
Υποστηρίζει Σκούρο και Ανοιχτό θέμα.
